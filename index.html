2019 Forbes Global 2000

The Forbes Global 2000 is an annual ranking of the world's top 2000 best performing public companies by the Forbes magazine. The ranking is determined using data in four metrics: revenue, profit, asset value and market value. The list has been published every year annually since 2003.

While many people use this ranking list as an indication of the world's leading public companies and their performances, it is not definitive and is only an interpretation. It should only be used as a general reference and not as an actual decisive evidence or data, since any changes to the criteria can change the ranking.
